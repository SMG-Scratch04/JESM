## 何ができるのか
最新地震情報1件+過去配信された地震情報19件を閲覧できます。※震度マップは現時点では最新地震情報以外見れません。

震度速報・震源情報・各地の震度の情報は受信できます。(確認済み)

その他も多分受信できます。(未確認)
